Streaming-pi
============

Librerias, ejemplos y apps para ser ejecutadas en Raspberry Pi.
>
**Proyectos**
 - Streaming
  - StreamingPiServer: Solo debe de funcionar en raspberry pi
  - StreamingPi (Cliente): Solo debe de funcionar en el PC (Windows/Linux/Mac).
